1. GlasAnketa — Employee Satisfaction System

Secured Anonymity Data-Collection Infrastructure
.NET 8.0 EF Core SQL Server

System Architecture & Intent

Developed an internal survey management web infrastructure allowing organizational leadership to deploy target questionnaires to track operational satisfaction trends. Built around strict data processing constraints, the pipeline encapsulates a robust session check to validate access metrics while keeping response metrics independent to guarantee anonymity.

Core Structural Entities

  • User: Holds demographic buckets (Age, Experience, Team/OU mapping) and security profiles.
  • QuestionForm & Question: Operates dynamic sequencing forms supporting Scale (1–10) and Text evaluation inputs.
  • Answer & AnswerSummary: Transactional logs evaluating aggregation targets per organizational group.

Transactional Controller Pipeline

[HttpPost]
[ValidateAntiForgeryToken]
public async Task<IActionResult> SubmitAnswers(FormSubmissionVM model)
{
    var userId = HttpContext.Session.GetInt32("UserId");
    if (userId == null) return RedirectToAction("Login");

    await _answerService.SubmitAnswersAsync(userId.Value, model.QuestionFormId, answers);
    var nextForm = await _questionFormService.GetNextActiveFormAsync(model.QuestionFormId);
    
    return nextForm == null 
        ? RedirectToAction("ThankYou") 
        : RedirectToAction("ShowForm", new { formId = nextForm.Id });
}

2. TruckFlow — Loading & Destination Tracker

Contract Management & Intelligent Transporter Selection Routing Engine
AutoMapper 16 LINQ Engine SQL Server 2022

System Architecture & Intent

An enterprise tracking application managing contract conditions, dispatch notes, and transport workflows. Features an automated business service logic matrix that reads live carrier parameters to instantly identify and route freight to the transporter holding the absolute lowest active contract price for any destination. Includes background task handlers monitoring a strict 2-day SLA window to execute auto-fallback re-routing rules if plate numbers are delayed.

Core Structural Entities

  • Transporter & Contract: Binds third-party shipping carriers to active legal terms and operational limits.
  • Condition & Destination: Tracks fixed pricing profiles, city enum parameters, and dynamic fuel adjustment values.
  • Order Management: Governs dispatch workflows from creation, through Pending plate registration, to Finished delivery manifests.

Algorithmic Best-Price Matching Engine

public async Task<Transporter?> GetCheapestTransporterAsync(int cityId, decimal weight)
{
    var validConditions = await _context.Conditions
        .Include(c => c.Contract).ThenInclude(con => con.Transporter)
        .Where(c => c.CityId == cityId && c.Contract.IsActive && c.Contract.EndDate >= DateTime.UtcNow)
        .ToListAsync();

    return validConditions
        .OrderBy(c => c.CalculateFormulaPrice(weight))
        .Select(c => c.Contract.Transporter)
        .FirstOrDefault();
}

3. Comet — Non-Conforming Output (NCO) Procurement

B2B Material Clearinghouse & Claims-Based Auction Portal
Claims Authentication Excel Bulk Ingestion Clean Architecture

System Architecture & Intent

A specialized B2B commerce and bidding framework designed to safely liquidate internal Non-Conforming Output (NCO) industrial components. The platform utilizes an Excel binary stream parsing infrastructure to allow administrators to mass-upload material sheets. Registered buyers participate in active bidding pools protected by strict transaction concurrency safeguards and security checks managed via .NET cookie identity schemas.

Core Structural Entities

  • Product Model: Monitors NCO lot entries, weight records, structural specifications, and reserve starting prices.
  • Bid Log Matrix: Records secure customer bids matching timestamp indexes to safeguard financial validation tracking paths.
  • User / Role Schemas: Grants restricted views dividing management upload panels from public buyer auction layouts.

Real-Time Bid Validation Strategy

public async Task<bool> ProcessOfferAsync(int buyerId, int productId, decimal offerAmount)
{
    var product = await _productRepository.GetByIdAsync(productId);
    if (product == null || product.IsAuctionClosed || DateTime.UtcNow > product.EndDate) 
        return false;

    if (offerAmount <= product.CurrentHighestBid) 
        return false;

    await _productRepository.UpdateHighestBidAsync(productId, offerAmount, buyerId);
    return true;
}
